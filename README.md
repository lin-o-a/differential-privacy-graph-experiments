# differential-privacy-graph-experiments
Experiments to optimise graph privacy protection and utility using stratification, flat crowd blending, and other techniques.
Graph datasets:
1. Synthetic graph of patients with typical and rare diseases (200 nodes in the graph)
2. Cora citations (2708 nodes)

Folders
1. Stratification - Experiment 1 with 1 synthetic graph
2. FlatCrowdBlending - Experiment 2 with 2 graphs(synthetic medical graph and Cora)
